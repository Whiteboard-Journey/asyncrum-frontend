Customization

Hyper is highly customizable and provides an easy way to change the layouts, color themes etc. Hyper comes with different layout choices, including vertical, horizontal, detached side menu, full mode etc. The layout and left sidebar related customization can be done by simply changing the value of a few state variables. Everything is being controlled through Redux. Please refer to below described details in order to see what are the possible customization or changes;

You can customize the look and feel of various elements, including navbar, topbar, form elements etc by simply editing Sass files in /src/assets/scss directory or removing unneeded components from app.scss source files.

1. How to change the default layout from vertical to another?

Open src/redux/layout/reducer.js file, and search for declaration of INIT_STATE. Here as a default layoutType property, you can either specify layoutConstants.LAYOUT_HORIZONTAL, layoutConstants.LAYOUT_DETACHED or layoutConstants.LAYOUT_FULL. The respective layout would get activated and being displayed as a default to user.

Hyper Default - Vertical Layout

This is already set as a default. You don't need to do anything.

Horizontal Layout

Specify default state parameter INIT_STATE's layoutType property with layoutConstants.LAYOUT_HORIZONTAL

Detached Sidebar Layout

Specify default state parameter INIT_STATE's layoutType property with layoutConstants.LAYOUT_DETACHED

Full Layout

Specify default state parameter INIT_STATE's layoutType property with layoutConstants.LAYOUT_FULL

2. How to change the default color scheme from light to dark?

Open src/redux/layout/reducer.js file, and search for declaration of INIT_STATE. Here as a default layoutColor property, specify layoutConstants.LAYOUT_COLOR_DARK in order to enable dark mode.

Dark Vertical Layout

Dark Horizontal Layout

3. How to enable boxed mode?

Open src/redux/layout/reducer.js file, and search for declaration of INIT_STATE. Here as a default layoutWidth property, specify layoutConstants.LAYOUT_WIDTH_BOXED in order to enable boxed mode.

Boxed Vertical Layout

Boxed Horizontal Layout

4. How to change the left sidebar theme?

The left side bar (mainly available in Vertical and Detacehd layouts) can be changed very easily as well. There are three possible choices of styles:
a. Default - the default side bar with purple background
b. Light - the sidebar with light background
b. Dark - the sidebar with dark background

Open src/redux/layout/reducer.js file, and search for declaration of INIT_STATE. Here as a default leftSideBarTheme property, specify layoutConstants.LEFT_SIDEBAR_THEME_DEFAULT or layoutConstants.LEFT_SIDEBAR_THEME_LIGHT or layoutConstants.LEFT_SIDEBAR_THEME_DARK in order to activate respective sidebar theme.
