Themes

Hyper comes with multiple overall color schemes. Currently there are three themes available: Saas (Default), Creative and Modern. Each of these are available in dark mode as well. You can switch to any of them by simply making one line change in the code.

Open src/App.js file and on top of the file, simply import the respective scss file. e.g. search for "Saas.scss". Here you can simply uncomment import statement for modern or creative. Make sure to comment the import statement of other themes, otherwise your bundle would have css compiled for all the themes.

Following table is containing file list for each respective theme/demo:

Theme/Demo Scss File
Saas (Default) /assets/scss/Saas.scss
Modern /assets/scss/Modern.scss
